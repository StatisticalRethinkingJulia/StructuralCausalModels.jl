Introduction

These are some very, very early tests to see what it would take to create a Julia version of the SCM primer book.

Important links are:

  1. Dagitty
  2. R dagitty package

and particularly:

  1. R ggm package

My initial goal for this package to have a minimal way to apply SCM ideas to the examples in StatisticalRethinking.jl.

A secondary goal is to have a structure to hold model info. Currently I'm experimenting with this package, with MCMCModels.jl and MCMCParameters.jl.

This package is not registered and might never be!!!!
